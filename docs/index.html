genKey

Generates a new PGP key pair. Supports RSA and ECC keys. Primary and subkey will be of same type.

genKey(data: Object, callback: function (err, key)?): Promise<(keyData | Error)>
Parameters
data (Object) Information about the new pair of keys.
Name Description
data.name String Public name to use for keys creation.
data.email String Public email to use for keys creation.
data.pass String Secret password to generate the private key.
data.numBits Number (default 2048) Define the RSA key size used for generate the private key. Posibble values (4096 or 2048).
data.curve String (default 'curve25519') Define the curve used for generate the private key. Posibble values (curve25519, p256, p384, p521, secp256k1, brainpoolP256r1, brainpoolP384r1, brainpoolP512r1).
callback (function (err, key)?) A callback to run the error and new key
Returns
Promise<(keyData | Error)>: A promise for the user's keys
